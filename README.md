# Epdf
PDF editor written in rust, under development

- --help: 显示帮助信息
- --dis 显示pdf ：
    - .\Epdf.exe --dis concat-test.pdf
- --copy 新建pdf ：
    - .\Epdf.exe --copy concat-test.pdf 1-2,2,2,2,2,1,1,1
- --water 加水印 : 
    - --water concat-test.pdf waterMark
- --2images 转图片: 
    - --2images concat-test.pdf
